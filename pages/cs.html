  • Beginning

    Basic Concepts

    Code.org

    Well-known for its Hour of Code activites that are a low-commitment way to pique the interest of children, Code.org features not only quick activities, but also full-length curriulums for grades K-12. Its mission is to expand access to computer science in schools and increase participation by young women and students from other underrepresented groups. Their curriculums feature videos and interactive activities with visuals for students to learn basic programming concepts, HTML, CSS, JavaScript, and more.

    Scratch

    Scratch was designed to target children from ages 6 to 16, but can be used by people of all ages for free. It uses block-based programming to teach concepts such as event listeners, loops, conditionals, operators, variables, and even functions! If you're not sure where to start, Scratch also features tutorials and projects created by other users that can give you some inspiration. Children particularly enjoy customizing sprites, backdrops, and sounds in order to create interactive games and stories.

    Alice

    Alice is a block-based programming tool available to download for free that can be used to create animations, build interactive narratives, and program simple games in 3D. Named after the character in Alice’s Adventures in Wonderland, the goal of this team is to make computer programming fun and easy, just as Lewis Carroll made his books clear and entertaining. Alice focuses on object-oriented programming, and a study on an undergraduate programming course using Alice 3 and Java saw an increase in at least a letter grade when Alice was used as an intermediary step before Java, as opposed to solely using Java.

  • Intermediate

    Writing Your First Code

    Particularly if you already have experience with programming concepts from block-based programming or any other way of training to "think like a programmer," learning your first language shouldn't be too difficult! However, particularly if you do not have any prior experience (and even if you do), it's a good idea to follow a tutorial or other curriculum to learn your first langauge. A good first language might be Python, although Java is commonly used in high schools, and JavaScript could also be a good starting point. At this stage, if you are interested in building websites, it may also be a good idea to look into learning the basics of HTML, CSS, and JavaScript.

    Curriculums

    High schools often offer introductory programming courses. Some also follow College Board's AP Computer Science A and AP Computer Science Principles courses, which have ready-to-use curriculums you can follow along with from endorsed providers. Besides high schools, many colleges have also published their introductory course materials. Harvey Mudd College has a publicly (and seasonally) available introductory course in Python. Harvard is known for its CS50x introductory course for everyone, which covers a variety of languages. The course is available on edX for free.

    freeCodeCamp

    freeCodeCamp is a (yes, you guessed it!) free website that has fantastic guided tutorials you can follow along with. You can learn Python in their "Python for Everybody" course. Additionally, if you are interested in web development, you can learn about HTML and CSS in their "Responsive Web Design" course and JavaScript in their "JavaScript Algorithms and Data Structures" course.

    Khan Academy

    Khan Academy offers introductory course featuring their well-known structure of videos, articles, and practice. They have Hour of Code lessons, but also HTML, CSS, and JavaScript tutorials, resources for general knowledge information about computers and the Internet, and a bit on more advanced topics like algorithms, cryptography, and information theory.

    Miscellaneous

    Besides online free resources, there are many companies and individuals who offer tutoring services for computer science. If it's your first time learning a language, don't hesitate to reach out for help! It will ensure that you have a solid foundation in case you want to learn more advanced topics, which is crucial.

    Opportunities for High School Students

    Clubs

    If you are lucky enough to have a high school with computer science related clubs, be sure to take advantage of them. If not, don't worry about it. Instead, see if you can meet up with some of your friends and make a fun project together!

    Summer Camps

    There are a plethora of summer programs for high schoolers interested in computer science. Many are hosted at universities, so be sure to see if a higher education institude near you has any offerings. Here are a few:

    • Google's CSSI is a 4-week introduction to computer science for rising college freshmen with a passion for technology - especially students who’ve been historically marginalized from the field.
    • CodeConnects is an initiative of The Coding School, which aims to empower the next generation through computer science education. It offers 1:1 lessons, summer camps, and more. They also have scholarships if you demonstrate financial need!
    • Upperline Code offers introductory and more advanced Python and JavaScript courses for students depending on their experience level. It also has a limited number of scholarships available.

    And some targeted towards females:
    • Girls Who Code offers 2-week virtual summer programs for rising 10-12th-grade girls to learn coding and get exposure to tech jobs.
    • Kode with Klossy targets female-identifying and non-binary individuals ages 13-18. They have 2-week summer programs and 3 different curriculums, but no experience is necessary for any of them!
    • Tech Scholars has a mission to bridge the gender gap in technology, and to make coding accessible to girls. Its weeklong "Web Design" course is free for high school girls nationwide.

  • Advanced

    Languages

    Core Languages

    After learning one language, it tends to be easier to pick up others. You already know the concepts, and now you just have to become attuned to syntax differences. Here are some languages that are probably useful to learn if you aren't already familiar with them:

    • Python has relatively simple syntax, and has many popular packages for artificial intelligence, data science, and financial services.
    • HTML, CSS, and JavaScript are the building blocks of front-end development. HTML designs the structure, CSS controls the styling, and JavaScript allows for animations and more complicated functionality.
    • Java is frequently used in large companies, and can also be used to develop Android applications.
    • C, along with Python and Java, forms a solid foundation for programming as one of the "core languages." Its syntax is a bit more complex because it is older, but it can run on any device so it is often used to program hardware. C# is Microsoft's variant of C, intended to be faster and more secure. C++ is an extension of C, used for everything ranging from computer games to mathematical simulations.

    More Languages

    There are many other languages that are a bit more specialized. Depending on your interests, you may want to investigate further into one or several of them.

    • Kotlin is often used to program Android applications.
    • Swift is used to program iOS and macOS (Apple) applications.
    • Go or Golang is used in applications that need to process a lot of data (in companies like Google, Netflix, Twitter, and Uber), but is not broadly used.
    • R is heavily used in statistics.
    • PHP is a scripting language like Python, widely used for server-side web development. It is an older language used by companies like Facebook, Yahoo, and Wikipedia (and more than 80% of websites on the Internet!), but its popularity is dropping as JavaScript and Python have started replacing it. Ruby is also popular for web development that is relatively easy to prototype in.
    • MATLAB, Mathematica, and Julia are used for scientific research and scientific computing. MATLAB is older and more established, while Julia is newer but increasing in popularity.

    Frameworks and Other Tools

    Frameworks

    • React and Angular are the most popular JavaScript frameworks.
    • Node.js allows you to run JavaScript on the server-side, even though it was traditionally used as a client-side scripting language.
    • jQuery is another JavaScript framework for client-side scripting. You can do animation, send HTTP requests, reload pages, and perform client-side validation by writing just a couple of lines of code.
    • React Native and Flutter are used for cross-platform (Android and iOS) applications.
    • Bootstrap provides HTML- and CSS-based templates.
    • Firebase is Google's mobile platform that helps you quickly develop high-quality mobile apps and grow your business.
    • Django and Flask are popular Python frameworks.

    Version Control

    Git is a version control system widely used in industry that is very important to learn about.

    Google!

    Very soon (if you haven't already), you will discover that you do not, in fact, have the all the coding knowledge in the world. Make sure to leverage Google for resources like Stack Overflow to see if other people have had similar errors as you.

    Linux

    Being comfortable with the Linux command line allows you to work effectively. You can search files, know what’s going on with the system by checking its CPU and memory usage, and many other perform basic and advanced tasks.

    Cloud Computing

    Cloud computing concepts and platforms like Amazon Web Services (AWS), Google Cloud Computing (GCP), Microsoft Azure are becoming ever-more important.

    Databases

    There are many databases, but knowing one will get you started. It's a good idea to know how to insert, update, and delete data and write SQL queries to retrieve it.

    Containers

    Docker and Kubernetes provide you with tools for deployment, so that you don't have to worry about your application's scalability.

    Independent Practice

    Hackathons

    As you're exploring new languages and frameworks, hackathons (check MLH or your local colleges and universities) offer a great chance to explore in just a weekend. Enlist your friends, or make new ones while enjoying free food!

    Personal Projects

    It's also important to have some projects that you've invested more time in. Is there a game you want to replicate? Try making a personal website, or a todo app. Play around with web and app development, machine learning and artificial intelligence, and anything else that strikes your interest.

    Data Structure and Algorithms

    A cornerstone of any technical interview with a larger company, solid knowledge of data structures (linked lists, graphs, arrays, trees, etc.) and algorithms (searching and sorting, dynamic programming, greedy algorithms, runtime analysis, etc.) are a must-know! To practice, LeetCode has classic problems, and Cracking the Coding Interview is also a popular guide.
